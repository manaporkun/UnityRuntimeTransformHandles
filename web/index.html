Runtime Transform Handles

Unity runtime gizmo system with WebGL demo.

This page prioritizes the playable build on the left and keeps the project docs on the right for quick evaluation.

About

Based on the original RuntimeTransformHandle project by pshtif, this package extends runtime manipulation with multiple selection, configurable snapping, origin modes, and improved auto-scaling.

Core Features

  • Position, rotation, and scale handles
  • Multiple object selection and manipulation
  • Pivot and center origin modes
  • World and local coordinate spaces
  • Keyboard shortcuts and interaction events

Default Shortcuts

Key Action
WPosition mode
ERotation mode
RScale mode
AAll modes
XToggle World/Local
ZToggle Pivot/Center
